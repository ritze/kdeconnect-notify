# kdeconnect-notify

```
usage: kdeconnect-notify.py [-h] [--debug] [-d DEVICE_ID] [-n DEVICE_NAME]
                            [--hide-notifications] [-q]

Display notifications of your android device over KDEConnect.

optional arguments:
  -h, --help            show this help message and exit
  --debug               print debug messages
  -d DEVICE_ID, --device-id DEVICE_ID
                        set device id
  -n DEVICE_NAME, --device-name DEVICE_NAME
                        set device name
  --hide-notifications  hide notifications
  -q, --quiet           hide terminal messages
```
